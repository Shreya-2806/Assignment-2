W3C

In October 1994, Tim Berners-Lee founded an organisation called the World Wide Web Consortium(W3C)-devoted to developing non proprietary interportable technologies for the World Wide Web

Web 2.0

In 2003, there was a noticeable shift in how people and business were using the web and developing web-based applications. The term Web 2.0 was coined by Dale Doughterty of O'Reilly Media in 2003 to describe this trend.
